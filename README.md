📚 Skill Swap
A peer-to-peer learning platform where users can teach skills and learn new ones by exchanging knowledge — no money, just skills.

🚀 Features
🔐 User Authentication (Login / Signup)

👤 User Profiles (skills you teach & learn)

🔍 Skill Matching System

💬 Realtime Chat

📅 Session Booking

⭐ Ratings & Feedback

🛠 Tech Stack
Frontend: HTML, CSS, JavaScript (or React)

Backend: Firebase (Auth, Firestore, Hosting)

Version Control: GitHub
